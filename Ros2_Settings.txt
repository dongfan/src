Ubuntu 설치 이후 Ros2 환경설정 (Ubuntu 버젼 22.04)

1. Terminator 설치
sudo apt update    //apt(Advance Packging Tools)

/*
update - 설치 가능한 패키지 리스트를 최신화
upgrade - 실제 업데이트
*/

sudo apt install terminator //터미네이터 설치

2. Chrome 설치
sudo apt update
sudo apt upgrade

wget --version //설치유무와 버젼 확인

sudo apt install wget //wget 설치

wget https://dl.google.com/linux/direct/google-chrome-stable_current_amd64.deb //크롬 다운로드
sudo dpkg -i google-chrome-stable_current_amd64.deb //크롬 설치

sudo apt install chromium-browser

8. bashrc 설정하기
echo $SHELL // 현대 SHELL 확인
(echo 명령어는 python의 print 와 같은 역할)

subl ~/.bashrc //Sublime으로 .bashrc 열기
(bashrc : 터미널 실행시 가장먼저 실행하는 설정을 보관하는 파일)

9. alias 설정하기
alias //터미널에서 치면 alias 명령어 목록이 나옴

alias sb="source ~/.bashrc; echo \"bashrc is reloaded\""
alias ros_domain="export ROS_DOMAIN_ID=13; echo \"ROS_DOMAIN_ID=13\""
alias humble="source /opt/ros/humble/setup.bash; ros_domain; echo \"ROS2 humble is activated!\""
alias turtle="ros2 run turtlesim turtlesim_node; echo \"Ros2 Turtlesim Run!\""
